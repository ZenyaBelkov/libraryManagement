# Library Management System

This project is a simple library management system where users can add, delete, find, and update books. The data is stored in a JSON file.

## Features:
- **Add a book**: Add a new book with a title, author, year of publication, and status.
- **Delete a book**: Remove a book from the system by its ID.
- **Find a book**: Search for a book by title, author, or year.
- **Show all books**: Display a list of all books in the library.
- **Update book status**: Change the status of a book between 'Available' and 'Borrowed'.

## How to Use:
1. Clone the repository or download the files.
2. Run the `library.py` script.
3. Follow the on-screen prompts to manage the library.

## Requirements:
- Python 3.x
- json module (comes pre-installed with Python)
